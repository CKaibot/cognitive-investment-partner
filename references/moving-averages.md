# 《均线的艺术》学习笔记
## 一、均线核心基础认知
1. 核心本质
   - 内容：均线是市场的平均持仓成本线，短期均线拐头代表短期资金成本变化，长期均线走平代表长期筹码沉淀。趋势的本质是均线的排列方向：短期均线在上、长期均线在下、依次向上发散为多头排列，对应上升趋势；反之是空头排列，对应下跌趋势。
   - 原话："均线不是拿来猜涨跌的，是拿来给你算市场里所有人的平均持仓成本的，成本决定了抛压和支撑，这是核心"[02:12-03:05]
2. 有效跌破/突破的量化标准
   - 内容：有效跌破=收盘价低于对应均线3%+连续3个交易日收在均线下方，两个条件缺一不可，否则为假摔洗盘；有效突破=收盘价站在均线上方+连续3个交易日不跌破+量能达标。
   - 原话："什么叫有效跌破？收盘价低于你看的那条均线3%，而且连续3天收在下面，缺一不可，不然就是假跌破，是洗盘，不要慌"[04:42-04:55]
3. 均线与量能的绑定关系
   - 内容：所有均线信号必须结合量能判断：无量的突破是假突破，无量的回踩是真支撑，放量的跌破是真破位，放量的反弹是诱多。单独使用均线准确率仅50%，搭配量能后准确率可提升至70%以上。
   - 原话："均线只是工具，不是圣杯，你要结合量能、位置、板块情绪一起看，单独用均线准确率只有50%，搭配量能可以到70%以上"[48:28-50:15]
   - 原话："量和均线是亲兄弟，缺一不可，无量突破的，你看10个有9个是假的，放巨量跌破均线的，你别等反弹，直接走，那就是真出货了"[48:42-49:05]

## 二、均线核心形态及交易规则
### （一）均线回踩形态
1. 形态构成条件
   - 标的处于上升趋势，至少20日、60日均线呈多头排列向上；
   - 回调阶段量能逐步缩量，不能放巨量下跌；
   - 通常回踩20日线（短线）、60日线（中线），回踩时K线下影线触碰均线即拉回，未触发有效跌破条件。
2. 操作规则
   - 买点：回踩确认后次日收阳时进场；
   - 止损：设置在回踩最低点下方2%位置。
3. 案例：2023年10月新能源板块走势，回踩60日均线时缩量，连续3天收盘价未跌破60日线，之后反弹幅度达28%，完全符合该规则。[05:35-06:12]

### （二）均线突破形态
1. 形态构成条件
   - 突破标的为长期均线（60日、120日、250日）；
   - 突破时量能达到前20个交易日平均量能的1.5倍以上；
   - 突破后连续3个交易日收盘价站在均线上方，回踩不跌破均线；若为跳空突破且缺口不回补，是最强的突破信号。
2. 操作规则
   - 买点：突破当天分时走势走稳时进场，或回踩均线确认后进场；
   - 止损：设置在突破的均线下方3%位置。
3. 案例：2023年2月中航电测，突破250日均线时量能放大3倍，之后连续一字板，涨幅超300%。[14:15-14:52]
4. 原话："突破长期均线的时候，量能必须至少比之前20天的平均量放大50%，不然就是假突破，你进去就被套，很多人买突破亏就是因为没看量"[12:58-13:15]

### （三）均线背离形态
1. 形态构成条件
   - 顶背离：股价创新高，但对应的短期均线（5日、10日）未同步创新高；仅在标的涨幅超过30%的高位出现有效，连续2次顶背离基本确认阶段见顶。
   - 底背离：股价创新低，但对应的短期均线（5日、10日）未同步创新低；仅在标的跌幅超过40%的下跌末端出现有效，连续2次底背离基本确认阶段见底。
2. 操作规则
   - 顶背离：股价跌破10日线时止盈离场；
   - 底背离：股价站上20日线时进场抄底。
3. 案例
   - 2022年4月宁德时代走势，股价创新低但10日线未同步创新低，底背离之后反弹幅度超50%；
   - 2023年8月AI板块走势，股价创新高但5日线拐头向下未创新高，顶背离之后回调幅度达35%。[08:50-09:28]
4. 原话："顶背离一定要出现在涨幅够大的地方，涨了5%就顶背离那是骗你的，没有30%以上的涨幅，顶背离没有意义，底背离同理，没跌30%以上的底背离都是下跌中继"[07:55-08:18]

### （四）均线粘合形态
1. 形态构成条件
   - 短中长均线（5、10、20、60日）粘合在一起，各均线间差值小于5%，波动幅度小于10%，持续时间超过1个月，代表市场平均成本趋于一致，即将变盘；
   - 粘合后放量向上发散为多头行情启动信号，向下发散为空头行情启动信号，向上发散时需满足5日线上穿10、20、60日线，同时量能放大2倍以上。
   - 粘合时间越长，后续行情幅度越大：粘合3个月对应翻倍行情，粘合半年对应数倍级别行情，即"横有多长竖有多高"。
2. 操作规则
   - 向上发散确认后第一时间进场；
   - 向下发散确认后立刻清仓离场。
3. 案例：2023年上半年中际旭创，均线粘合40天后向上发散，3个月涨幅超300%。[23:50-24:30]
4. 原话："均线粘合的时间越长，后面的行情越大，粘合3个月的，涨起来就是翻倍，粘合半年的，那就是几倍的大行情，横有多长竖有多高，说的就是均线粘合的情况"[22:45-23:08]

### （五）金叉死叉的误区修正
1. 判断规则
   - 上升趋势中，首次5日与10日均线死叉通常为洗盘信号，只要20日均线仍保持向上，后续很快会再次金叉，属于低吸点而非卖点；
   - 仅当高位20日均线拐头向下后出现的死叉，才是真正的见顶卖点。
2. 案例：2023年10月贵州茅台，上升趋势中首次出现5、10日线死叉，20日线保持向上，之后很快金叉反弹15%。[34:15-34:48]
3. 原话："上升趋势里的首次短均死叉是洗盘，你要是卖了就飞了，只有当均线的大方向拐头了，死叉才是真的见顶"[32:45-33:02]

### （六）均线发散强度判断
1. 规则：均线向上发散的角度越大，短期行情强度越高；45度角向上发散是最健康的走势，行情持续性最强；角度超过60度代表短期涨幅过快，容易快速见顶；角度小于30度代表行情太弱，参与性价比低。
2. 原话："均线发散的角度，45度是最好的，不紧不慢，走的最远，那些连续涨停的，角度超过60度的，涨的快跌的也快，你要是没那个本事就别碰，赚你看得懂的钱"[1:42:32-1:44:18]

## 三、均线周期搭配用法
1. 短线（持仓1-10天）：使用5、10、20日均线组合，核心参考20日均线，20日线向上可参与短线操作，跌破20日线立刻离场。
2. 中线（持仓1-6个月）：使用20、60、120日均线组合，60日线是中线生命线，120日线是牛熊分界线，60日线与120日线金叉后代表中线行情启动。
3. 长线（持仓6个月以上）：使用60、120、250日均线组合，250日线（年线）走平向上是长线牛市的基础，年线向下时不建议长期持股。
4. 原话："短线看20日，中线看60日，长线看年线，你做什么周期就看什么均线，不要拿着短线的信号做长线，那是找死"[18:22-20:05]
5. 原话："20日线是短线的命根子，只要你做短线，20日线拐头向下，不管你赚亏，立刻走，不要等，等就是深套，我做短线这么多年，20日线破了不走的，9成要亏"[18:47-19:12]

## 四、不同市场环境下的均线应用
1. 熊市应用规则：熊市中60日线通常向下，所有反弹到60日线的走势都是卖点，该位置是套牢盘集中解套的位置，压力极大，不要追高；仅当60日线走平向上后，才可转变思路开始做多。
2. 原话："熊市里你就记住，碰60日线就卖，准没错，什么时候60日线抬头了，你再回来买，能避开90%的下跌"[3:22:40-3:25:10]
3. 原话："2022年我就是靠这个规则，躲开了所有的大回调，全年还赚了15%"[3:22:52-3:23:05]
4. 趋势核心原则：趋势一旦形成，均线方向不会轻易改变，绝对不要逆均线方向操作，均线向上就做多，均线向下就空仓或做空，顺势而为是盈利的核心。
5. 原话："顺均线者昌，逆均线者亡，均线向上你就做多，均线向下你就休息，就这么简单，你非要在均线向下的时候买，那就是和市场对着干，亏了是你自找的"[3:55:05-3:57:20]

## 五、配套战法及实盘纪律
### （一）20日均线短线战法（回测成功率72%）
1. 选股条件：20日均线向上，股价在20日均线上方，最近10个交易日出现过涨停，股性活跃；
2. 买点：股价缩量回踩20日均线，未有效跌破，次日收阳线、分时均价线向上时进场；
3. 卖点：股价跌破20日线止损，或短线盈利10%-20%后，5日线拐头向下时止盈；
4. 回测数据：2023年全年用该战法操作活跃股，平均持仓5天，平均单次收益8%，成功率72%。
5. 原话："20日均线战法我用了10年，每年稳定30%以上的收益，就三个条件：20日线向上，缩量回踩不跌破，次日收阳，就这三个，做到你就能赚钱，难的是你守不守纪律"[1:28:10-1:30:25]

### （二）均线+黄金分割压力支撑计算法
1. 计算规则：
   - 压力位=均线价格+（前期高点-均线价格）*0.618
   - 支撑位=均线价格-（均线价格-前期低点）*0.618
2. 适用场景：适合判断反弹的压力位和回调的支撑位，准确率较高。
3. 原话："均线加黄金分割算压力支撑，准确率非常高，你们自己去复盘看看，很多股票反弹到60日线加0.618的位置就掉下来，就是这个道理"[2:14:08-2:16:30]

### （三）实盘操作纪律
1. 仓位规则：无论均线信号多么准确，永远不要满仓单只个股，最高仓位不超过7成，保留3成仓位作为风险缓冲。
2. 止损规则：止损必须严格执行，到达止损位立刻离场，不要抱有"会涨回去"的幻想，幻想是炒股最大的敌人。
3. 原话："不管信号多好，永远不要满仓一只股，最多7成，剩下的3成是你的后路，止损一定要严格，到了止损位就走，别幻想会涨回去，幻想是炒股最大的敌人"[10:09:15-10:12:30]

# 均线与均量线关系学习笔记
## 一、基础量价均线对应逻辑
### 1. 上升趋势量价匹配规则
[02:00-03:40]
- 核心观点：上升趋势中，均线的上行必须对应均量线同步上行，"价涨量增"是良性上涨结构，量价背离（均线创新高但均量线未同步创新高）是明确见顶信号。
- 量化条件：5日、10日、20日短周期均线拐头向上时，同期5日、10日均量线必须同步拐头向上，量能峰值跟随价格峰值逐步抬高。
- 操作规则：上升过程中若均量线提前拐头向下，即使均线仍保持向上，也要开始减仓，规避后续回调风险。

### 2. 下跌趋势量价匹配规则
[22:10-24:30]
- 核心观点：下跌趋势中均线向下时，若均量线持续缩量，说明下跌无承接，趋势将延续；若均量线先放量拐头，随后均线才拐头，是见底信号。
- 量化条件：下跌末期，均量线5日先上穿10日，持续放量3个交易日以上，且量能放大到下跌过程平均量能的2倍以上，后续量能未缩回到均量线金叉前水平，随后5日均线上穿10日均线，确认底部反转。
- 操作规则：均量线金叉时可试仓，均线金叉时加仓，股价跌破20日均线止损。
- 讲师原话："下跌的时候先看量，量先起来了，价格才会起来，量没起来的均线拐头都是反弹，不是反转。"
- 案例：2022年10月某新能源股底部，10月12日均量线金叉，连续3天放量2倍，10月18日均线才金叉，之后股价从12元涨到36元，涨幅200%。

## 二、典型趋势形态的量价均线判定标准
### 1. 台阶式上涨结构
[05:30-07:10]
- 形态构成条件：价格每上一个台阶（均线平台突破），对应的均量线也同步上一个台阶，每一轮上涨的量能峰值比前一轮高，回调时均量线同步缩量到前一轮启动时的水平，均线回踩支撑均线（如20日线）不跌破。
- 适用位置：股价上升趋势初中期，未出现过加速上涨的阶段。
- 操作规则：缩量回踩支撑均线时为买点，放量突破前高时加仓，均量线跟不上突破量能时为卖点。
- 讲师原话："台阶式的涨，就要配台阶式的量，量的台阶和价的台阶对得上，这个趋势就没走完，对不上就是假突破。"
- 案例：2023年某白马股走势，3次平台突破每次量能都比前一次高，趋势延续8个月，涨幅120%，直到第四次突破量能未跟上，随后见顶回调30%。

### 2. 均量线领先均线拐头的预警信号
[12:40-14:20]
- 量化条件：上升趋势中，均量线5日死叉10日的时间，比均线5日死叉10日的时间早3-5个交易日，是领先见顶信号；若均量线死叉后均线仍创新高，属于"量价顶背离"，信号可靠性极高。
- 适用位置：所有上升趋势末端，尤其是累计涨幅超过50%之后的高位区域。
- 操作规则：均量线先死叉时减一半仓位，等均线死叉确认时清仓。
- 案例：2024年某AI概念股顶部，均量线在6月12日死叉，均线直到6月18日才死叉，提前6天发出信号，这6天股价还创了新高，追高资金随后被套，股价从42元跌到18元。

### 3. 均量线均线同步金叉强势结构
[35:20-37:40]
- 形态构成条件：5日均量线上穿10日均量线的当天，或前后1个交易日内，5日均线上穿10日均线，形成同步金叉，同时MACD也在低位金叉，形成三共振。
- 适用位置：股价长期下跌后刚启动的位置，或上升趋势中回调到位的拐点位置。
- 量化条件：同步金叉当天的量能是前20个交易日平均量能的1.5倍以上，若当天涨停信号可靠性更高。
- 操作规则：同步金叉当天可重仓进场，止损放在金叉前的最近低点，止盈待均量线先死叉时减仓，均线死叉时清仓。
- 案例：2023年某券商股启动点，2023年4月21日，均量线和均线同步金叉，当天放量涨停，之后2个月涨幅110%。

### 4. 上升趋势回调到位判定（空中加油结构）
[48:10-50:20]
- 核心观点：上升趋势回调时，均量线缩量+均线撑住支撑位，是回调到位信号，后续大概率开启新一轮上涨。
- 量化条件：均量线缩量到5日均量线低于10日均量线，且量能萎缩到前一轮上涨峰值的30%以下，缩量持续3-5个交易日以上；均线回踩20日或60日长期支撑均线，收盘价跌破支撑位幅度不超过3%，且次日收回。
- 操作规则：此时可低吸，等均量线再次拐头向上、均线重新发散时加仓。
- 讲师原话："缩量踩均线，只要量缩到位，均线撑住了，后面大概率还有一波，这就是空中加油的结构。"
- 案例：2023年8月某消费股回调，均量线缩量到前期峰值的25%，回踩20日线当天收长下影，次日放量上涨，之后1个月涨幅50%。

### 5. 顶背离量化判定标准
[1:02:30-1:04:50]
- 量化条件：股价创出新高，5日均线也同步创出新高，但5日均量线的峰值比前一次价格高点对应的均量线峰值低30%以上，且10日均量线已经拐头向下，为明确顶背离信号，后续下跌概率超过90%。
- 操作规则：顶背离出现当天减仓，3个交易日内未补量就清仓。
- 案例：2024年某医药股顶部，股价创38元新高，均量线峰值仅为前高的58%，随后10个交易日下跌35%。

### 6. 下跌趋势反弹判定
[1:32:10-1:34:30]
- 核心观点：下跌趋势中，均量线未持续放量，仅单日放量后随即缩量，哪怕均线出现金叉也只是反弹，不是反转，反弹高度不会超过前一个下跌平台的压力位。
- 操作规则：该类情况只能快进快出，盈利10-20%就要离场，不能长期持有。
- 案例：2023年某地产股下跌过程中的反弹，均量线仅放量1天，之后持续缩量，均线金叉后反弹15%就继续下跌，后续再创新低，跌幅达30%。

### 7. 中长期大级别行情判定
[2:18:20-2:20:40]
- 核心观点：中长期均线拐头向上时，必须对应中长期均量线同步拐头向上，才说明是中长期资金进场，大级别行情可靠性更高；若中长期均线向上但中长期均量线向下，说明是短期资金拉抬，行情持续性差。
- 量化条件：60日均线上穿120日均线（形成银山谷/黄金交叉）时，60日均量线也要同期上穿120日均量线，形成量能长期金叉，后续行情涨幅通常超过1倍。
- 操作规则：出现这种长期同步金叉时可长期持有，直到60日均量线拐头向下时再考虑减仓。
- 案例：2019年某白酒股的长期金叉，均线和均量线的60日周期同时上穿120日周期，之后2年涨幅达8倍。

### 8. 三重顶/底的量价均线判定
[2:47:30-2:49:50]
- 形态构成：价格三次探顶/探底，对应的均线每次高点/低点逐步抬升/降低，但均量线的峰值/谷值逐步降低/抬升，就是确认的顶部/底部信号。
- 操作规则：三重顶的第三个顶出现、量能跟不上时清仓；三重底的第三个底量能抬升时重仓进场。
- 案例：2021年某新能源龙头的三重顶，三次价格创新高，量能一次比一次低，之后股价从69元跌到22元。

## 三、通用操作规则
### 1. 止损规则
[1:57:40-1:59:20]
- 任何基于均量线、均线的买点，买入后若均量线未按预期放量反而继续缩量，且均线跌破支撑位（如5日跌破10日），必须止损，止损幅度控制在5-8%，禁止扛单。

### 2. 核心总结
[3:12:10-3:14:20]
- 讲师原话："量是价的先行指标，均量线永远走在均线的前面，你把均量线和均线的对应关系搞懂了，就不会抄底抄在半山腰，逃顶逃在山脚下。所有的假突破、假反弹，你看量能都能看出来，量骗不了人。"
