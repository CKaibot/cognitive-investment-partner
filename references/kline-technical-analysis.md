## K线基础-1（约29分钟）

### 一、见底反转类形态

#### 1. 夹单形态 [02:17-03:03]
- **构成条件**：下跌趋势中，1根大阴线后接1根完全被大阴线实体包裹的小K线（阴阳均可），再接1根实体反包小K线、且收盘价站在大阴线实体中部以上的阳线，3根K线组成
- **适用位置**：下跌末端，前期累计跌幅≥15%
- **操作规则**：阳线收盘前介入，止损位设置在大阴线的最低点
- **讲师原话**："夹单是下跌见底胜率很高的形态，亏就亏几个点，破了大阴底就走，赚的话至少10个点起步"

#### 2. 五倍大阴止跌信号 [03:56-04:32]
- **构成条件**：下跌过程中出现1根实体长度是前面5-10根平均K线实体长度5倍以上的大阴线，随后1-2根K线收回大阴线实体的1/2以上，且带有下影线回踩大阴线底部后拉回；大阴线当日伴随放量（恐慌盘释放）
- **适用位置**：连续下跌后的低位
- **信号含义**：放量大阴是最后一跌，恐慌盘全部杀出，随后收回说明有资金承接
- **操作规则**：收回大阴半分位时建仓，止损在大阴线最低点

#### 3. 孕线（包含线）形态 [06:18-07:02]
- **构成条件**：
  1. 前一根为大实体K线（阴/阳，实体长度是近期平均K线的3倍以上）
  2. 后一根K线的最高价、最低价完全在前一根K线的实体范围内，实体长度不超过前一根大实体的1/3
- **信号分类**：
  - 下跌孕线：下跌末端大阴线接小K线，见底信号，后续出阳线反包小K线则确认
  - 上涨孕线：上涨末端大阳线接小K线，见顶信号，后续出阴线反包小K线则确认
- **操作规则**：下跌孕线确认后买入，止损在前大阴线最低点；上涨孕线确认后卖出
- **讲师原话**："孕线核心是位置，涨了一大截出孕线肯定要跑，跌了一大截出孕线才是机会，小阴小阳的孕线没用，必须是大实体抱小的才有用"

#### 4. 内困三日翻红形态 [06:34-07:10]
- **构成条件**：大阴线后连续2根小K线完全在大阴线实体内部震荡，第三根收放量阳线，收盘价突破大阴线的开盘价
- **适用位置**：下跌末端，前期跌幅≥20%
- **信号强度**：比普通孕线信号更强，洗盘更充分，见底确定性更高
- **操作规则**：第三根阳线收盘前买入，止损在大阴线最低点

#### 5. 三连阴缩量止跌 [16:32-17:28]
- **构成条件**：下跌过程中连续3根阴线，第三根阴线实体比前两根明显缩小，成交量较第一根阴线缩量≥30%
- **适用位置**：连续下跌后的低位，前期跌幅≥15%
- **操作规则**：后续第一根阳线收盘前介入，止损在第三根阴线的最低点
- **讲师原话**："三连阴缩量，没人卖了，后面出阳就是拐点，这个形态我用了十几年，胜率很高"

#### 6. 低位大长腿（长下影线）止跌 [19:02-19:58]
- **构成条件**：单根K线下影线长度是实体长度的2倍以上，上影线极短或无，当日盘中最大跌幅≥3%，最终收盘拉回，跌幅收窄或翻红
- **适用位置**：连续下跌后的低位，前期跌幅≥20%
- **操作规则**：当日尾盘轻仓介入，若后续2天不跌破下影线最低点再加仓，止损在下影线最低点

#### 7. 上升吞没形态 [20:50-21:46]
- **构成条件**：前一根为阴线，后一根阳线的实体完全包裹前一根阴线的全部实体（包括影线则信号更强），阳线实体越大越好
- **适用位置**：下跌末端，前期跌幅≥20%
- **操作规则**：阳线收盘前买入，止损在被吞没的阴线最低点
- **讲师原话**："吞没必须是大实体吞小实体，小阴小阳互相吞没用，没有参考价值"

#### 8. 阳包多阴反转形态 [17:56-18:42]
- **构成条件**：连续3根以上阴线下跌后，出现1根大阳线，实体反包前面至少3根阴线的实体，收盘价站在最近3根阴线的最高点之上，成交量放大
- **适用位置**：下跌末端，前期跌幅≥20%
- **操作规则**：大阳线收盘前买入，止损在大阳线的最低点

### 二、见顶反转类形态

#### 1. 下降吞没形态 [20:50-21:46]
- **构成条件**：前一根为阳线，后一根阴线的实体完全包裹前一根阳线的全部实体，阴线实体越大越好
- **适用位置**：上涨末端，前期涨幅≥30%
- **操作规则**：阴线收盘前卖出

#### 2. 高位十字星见顶 [16:02-16:30]
- **构成条件**：上涨末端，累计涨幅≥30%后，出现十字星K线（实体长度≤影线长度的1/3，上下影线都较长），当日成交量较前几日放大≥30%
- **操作规则**：次日收阴线时立刻卖出

### 三、趋势与突破类规则

#### 1. 下降通道突破规则 [07:54-10:28]
- **通道绘制**：上轨为历次反弹高点连线，下轨为历次下跌低点连线，通道内至少3个高点、3个低点确认
- **突破有效条件**：
  1. 阳线收盘价站稳上轨，且较上轨价格涨幅≥2%
  2. 突破当日成交量≥通道内平均成交量的1.5倍
  3. 突破后3天内不跌回通道内部
- **操作规则**：突破当日建仓，止损在突破前最近的回调低点
- **补充诱空判断**：大阴线跌破下轨后3天内快速收回下轨之上+放量 = 假摔诱空，也是买点
- **讲师原话**："下降通道突破必须带量，无量突破都是耍流氓，大概率是骗炮"
- **配套信号** [09:42-10:15]：下跌末端单日涨幅≥5%大阳线 = 资金进场明确信号，回调不破大阳线最低点可介入

#### 2. W底（双底）形态 [24:32-27:28]
- **构成条件**：
  1. 第一次下跌见底后反弹，反弹幅度≥8%
  2. 第二次下跌的低点不跌破第一次低点（允许1%以内假跌破，2天内收回）
  3. 第二次底部成交量比第一次底部缩量≥20%
  4. 第二次反弹突破颈线位，突破涨幅≥3%，成交量≥前期均量1.5倍
- **操作规则**：突破颈线位时买入，止损在第二个底部最低点
- **目标涨幅**：底部到颈线位垂直高度的1-2倍
- **讲师原话**："W底第二个底量越小越好，说明没人卖了，突破必须放量，不放量的突破都是假的"

#### 3. 平台突破规则 [28:32-29:04]
- **构成条件**：下跌后横盘震荡平台，时长≥10个交易日，震荡幅度≤15%
- **突破有效条件**：大阳线突破平台最高点，涨幅≥5%，成交量≥平台均量2倍，3天内不跌回
- **操作规则**：突破当日买入，止损在平台最低点
- **讲师原话**："横有多长竖有多高，平台磨的越久，突破后涨的越多"

#### 4. 缺口操作规则 [21:38-22:32]
- **下跌缺口**：3天内回补 = 恐慌性缺口（见底信号）；3天不回补 = 下跌中继缺口（还有≥10%跌幅）
- **上涨缺口**：3天不回补 = 强势突破缺口（继续持有）；3天内回补 = 弱势缺口（要减仓）

### 四、通用规则

#### 止跌K线验证规则 [22:36-23:12]
- 任何止跌K线（大长腿、孕线、吞没、夹单）都需次日验证：**次日不跌破止跌K线最低点且收红，才是有效信号**；次日直接创新低就是假信号
- **讲师原话**："不要看见下影线就冲，第二天不创新低还能涨，才是真的止跌，不然就是下跌中继"

#### 量价背离规则 [24:06-24:28]
- 上涨中股价创新高，成交量比上一次高点成交量小≥20% = 量价背离 = 见顶信号，要减仓
- 下跌中股价创新低，成交量比上一次低点缩小 = 缩量下跌，抛压未释放，不要抄底

## K线基础-2（约24分钟）

### 一、趋势强弱判断

#### 1. 20日均线强弱判断 [15:40]
- 股价在20日均线上方且均线向上 = 可操作区间，仓位可放到7成以上
- 股价跌破20日均线且均线拐头向下 = 风险区间，仓位降到3成以下甚至空仓

#### 2. 上证指数成交量活跃度判断 [58:10]
- 单日成交量 < 3000亿 = 行情低迷期，空仓或1成仓位做超短
- 成交量连续3天站在8000亿以上 = 行情活跃期，可重仓操作主线板块
- **案例** [01:02:20]：2024年2-3月上证成交量连续 < 3000亿，期间个股中位数跌幅12%

#### 3. 上涨量能健康度判断 [09:15]
- 量能维持在均量线1-1.5倍 = 健康
- 超过2倍 = 警惕回调风险
- 低于0.5倍 = 买盘不足，趋势要走坏

#### 4. 均线粘合突破信号 [02:08:10]
- **条件**：5/10/20/60日四条均线粘合超过10个交易日，之后放量突破（成交量≥前20日均量1.5倍），当天涨幅>5%
- **操作**：突破时买入，止损在粘合均线下方3%
- **案例** [02:15:40]：某新能源汽车股2023年11月四条均线粘合12天，放1.8倍量突破涨7.2%，之后2个月涨110%

#### 5. 缺口趋势判断 [01:28:30]
- 上升趋势向上跳空缺口3天不回补 = 强势突破缺口，后续涨幅≥缺口到前低距离的1倍
- 下跌趋势向下跳空缺口3天不回补 = 破位缺口，后续跌幅≥缺口到前高距离的1倍
- **案例** [01:32:15]：某半导体个股2024年3月向上跳空缺口3天不回补，缺口到前低20元，后续涨21元

### 二、顶部信号与卖出规则

#### 1. 高位涨幅+巨量见顶 [04:10]
- 一波上涨结构涨幅达3倍以上 + 高位放出阶段/上市以来最大量 = 顶部信号极可靠
- **操作**：立刻清仓，不要抱反弹幻想
- **案例** [12:05]：某新能源个股2022年4-8月涨幅3.2倍，高位放上市最大量，随后连续跌停

#### 2. 量价顶背离见顶 [22:30]
- 股价创新高，成交量比上一次同价位成交量低30%以上 = 明确顶背离，背离后10个交易日内大概率大幅回调
- **操作**：背离出现后3个交易日内逐步减仓
- **案例** [25:10]：贵州茅台2021年1-2月，股价创2627元新高时成交量比2020年12月同位置低42%，随后8个交易日跌20%

#### 3. 高位涨停后放量跌停 [01:10:15]
- 高位先涨停，第二天高开低走跌停，跌停成交量≥前日涨停成交量的2倍 = 明确出货
- **操作**：当天无论什么价位都要卖出，后续至少还有20%下跌空间
- **案例** [01:15:40]：某医药股2023年12月，高位涨停后第二天放量2.3倍跌停，随后10个交易日跌32%

#### 4. 龙头股见顶信号 [55:20]
- 市场总龙头当天换手率>40% + 长上影线（上影线>实体2倍）= 明确见顶
- **操作**：当天必须清仓
- **案例** [56:40]：2023年5月某AI龙头换手率47%，上影线是实体3倍，之后股价腰斩

#### 5. 补涨股见顶规则 [01:05:30]
- 补涨股涨幅达到龙头第一波涨幅的60% = 见顶信号，高度不会超过龙头的70%
- **案例** [01:08:10]：2023年AI龙头第一波涨210%，补涨股涨122%（≈60%）后见顶
- **讲师原话**："补涨股的高度不会超过龙头的70%，这是铁律"

#### 6. 高位连板炸板出货 [01:45:20]
- 高位连板炸板，炸板成交量>全天成交量40%，尾盘不能回封 = 出货，当天卖出

#### 7. 无利空大幅低开 [01:22:10]
- 无利空消息突然跳空低开>5%，开盘10分钟没回补到低开幅度的一半 = 资金出货，立刻卖出

### 三、底部信号与买入规则

#### 1. 下跌趋势抢反弹 [07:25]
- **条件**：连续3根以上大阴线（单根跌幅>5%）+ 乖离率达-20%以上
- **操作**：小仓位（≤1成）试错，止盈在反弹到5日均线，止损在入场价-3%

#### 2. 下跌末端抄底规则 [32:15]
- **三个条件必须同时满足**：
  1. 连续下跌>3个月，累计跌幅>50%
  2. 底部出现连续2个涨停板，第二个涨停封单>流通盘1%
  3. 第二个涨停成交量比前一天放大不超过50%（抛压小，资金一致看多）
- **操作**：第二个涨停排队买入，止损在第一个涨停开盘价
- **案例** [38:45]：某AI概念股2023年10月-2024年1月累跌57%，连续两个涨停，第二个涨停量仅放大32%，随后1个月涨120%

#### 3. 十字星见底确认 [18:30]
- 下跌中出现十字星不能单独作为见底信号，必须等第二天收出涨幅>3%的阳线 + 量能比十字星当天放大30%以上
- **案例** [20:10]：某券商股2022年3月连续3个十字星，之后还跌15%，直到放量3%阳线才见底

#### 4. 龙头股回调低吸 [47:20]
- **条件**：市场总龙头第一次回调到10日均线 + 成交量连续3天递减 + 没有跌停板
- **操作**：买入，止损在10日均线下方2%，止盈看前高
- **案例** [51:30]：某算力龙头2023年3月回调到10日均线，连续3天缩量无跌停，买入后7天涨65%

#### 5. 板块第二波买点 [02:42:20]
- **条件**：主线板块第一次回调到板块指数20日均线 + 板块内至少3只个股连板（热度未散）
- **操作**：买入板块龙头，止损在板块20日均线下方1%
- **案例** [02:47:30]：2023年6月AI板块回调到20日均线，板块内4只连板股，买入龙头后10天涨80%

#### 6. 低位首板炸板买入 [01:45:20]
- **条件**：低位首板炸板，炸板成交量≤全天成交量20%，尾盘能回封 = 洗盘
- **操作**：第二天高开>3%可直接加仓
- **案例** [01:52:30]：某消费股2024年4月低位首板炸板，炸板量占比17%，尾盘回封，第二天高开4%加仓，后续5天4板

### 四、交易纪律与仓位管理

#### 止损铁则 [02:35:10]
- 任何一笔交易，亏损达到8%必须无条件止损，本金安全永远第一位

#### 仓位管理铁则 [02:52:10]
- 单只个股仓位最多3成，同板块个股仓位加起来不超5成，剩余仓位留着做T或补仓

### 五、讲师核心经验总结

> "所有的技术信号，位置比形态重要，同样的涨停，在底部和在顶部意义完全相反，不要光记形态不看位置，90%的技术亏损都是因为忽略了位置只看形态。" [02:28:45]

> "做交易不要追求完美，不要想卖在最高点买在最低点，只要吃到趋势中间的那段就够了，追求头尾的人最后都套在山顶或者抄在半山腰。" [02:18:20]

## K线基础-3（约46分钟）

### 一、趋势判断类

#### 1. 下降趋势反转判断 [23:12]
- 股价站稳60日均线 + 60日均线拐头向上 = 下降趋势正式反转
- 下降趋势中每次反弹都是卖点，只有趋势反转后才可参与
- **讲师原话**："不要去做下降趋势的股票，下降趋势的股票每次反弹都是卖点，你想着抄底，主力想着抄你家"

#### 2. 上升通道操作规则 [02:42]
- 回调到通道下轨 = 买点，涨到上轨 = 短期卖点
- 有效跌破下轨 = 趋势走坏，止损离场

#### 3. 短线趋势锚定（5日均线）[31:40]
- 5日线之上持股，有效跌破5日线（跌破>2%或收盘收不回）无论盈亏离场
- **讲师原话**："做短线一定要看5日均线，跌破5日均线不管赚亏都要走，短线纪律比什么都重要"

#### 4. 中长线趋势锚定（20日均线）[45:21]
- 沿20日均线持有，有效跌破20日均线再止盈止损

### 二、量价关系类

#### 1. 底部洗盘判断 [00:57]
- 第一次脱离底部区间时，放量拉升 + 回调缩量 = 正常洗盘，可持有

#### 2. 高位出货量价信号 [05:52]
- 涨幅>50%后高位出现连续长上影线 + 放巨量不涨（哪怕是阳线）= 主力出货，减仓或清仓

#### 3. 量价背离信号 [15:55]
- 股价创新高但成交量比前一波高点更小 = 量价背离 = 上涨动能不足，果断卖出
- **案例** [19:15]：贵州茅台2021年2月顶部，股价创新高但量能低于前一波高点，后续从2600跌至1500

#### 4. 横盘突破量能条件 [18:03]
- 上涨中期横盘量能逐步萎缩 → 横盘越久突破后涨幅越大（"横有多长竖有多高"）
- 注意：仅限上涨中期横盘，高位横盘突破警惕诱多

#### 5. 涨停突破有效性判断 [14:28]
- 有效突破：放量涨停突破，量能>平台均量50%，封单稳定不炸板 → 当日或次日回踩跟进，止损在平台压力位下方3%
- 假突破：量能未放大或反复炸板 → 不要追进

#### 6. 量能放大的位置含义 [35:10]
- 量能较前段均量放大>50%：低位放量 = 启动信号，高位放量 = 出货信号
- **讲师原话**："量能的高低是相对的，和前面一段时间的均量对比，不是和历史最高量对比"

### 三、K线形态与特殊走势类

#### 1. 二次上涨回调买点 [05:06]
- 第一波连续上涨后，第一次回调到10/20日均线，回调过程成交量持续萎缩，收出小十字星或小阴小阳止跌 = 二次上车买点
- 止损放在回调最低点下方2%
- **案例** [13:21]：宁德时代2023年10-11月上升通道中回调到20日均线缩量止跌，后续涨幅超30%

#### 2. V型反转成立条件 [08:10]
- 下跌末端连续急杀（跌幅≥20%），低位出现放量大阳线（涨幅≥7%或涨停，量能≥前日2倍），后续连续3天站稳大阳线收盘价 = V型反转成立
- 不满足 = 反弹诱多

#### 3. 下跌末端止跌判断 [09:40]
- **三个条件同时满足**：
  1. 低位出现止跌K线（十字星、锤头线、大阳线）
  2. 成交量连续两天放大，比下跌均量高30%以上
  3. 股价站稳5日均线3天不跌破
- **讲师原话**："抄底不要抄在半山腰，没有明确止跌信号之前，不要去接飞刀"

#### 4. 弱势反弹判断 [07:36]
- 下跌趋势中第一次急杀后的反弹，量能未有效放大 + 反弹到压力位（缺口/前期平台低点）就滞涨 = 弱势反弹，不参与

#### 5. 盘整区间操作 [22:40]
- 股价在10%-15%幅度区间震荡：下轨买、上轨卖
- 盘整>1个月且量能越来越小 → 向上突破 = 主升浪，向下突破 = 破位止损

#### 6. 不同位置跌停的含义 [24:35]
- 低位跌停（已跌>30%）：放量跌停后第二天反包大阳 = 主力暴力洗盘，后续看涨
- 高位跌停（已涨>50%）：不管放量缩量都是见顶信号，第一时间出局

#### 7. 跌停反包买点 [12:32]
- 反包阳线实体完全覆盖跌停阴线实体 + 量能>跌停当天1.5倍 = 反转信号，第二天可买入，成功率>70%

#### 8. 缺口判断规则 [28:45]
- 上涨初期向上缺口3天不回补 = 突破性缺口（强势，不用等回补直接买）
- 上涨末端向上缺口3-5天回补 = 衰竭性缺口（要卖）
- 下跌初期向下缺口 = 突破性缺口（赶紧止损）
- 下跌末端向下缺口 = 衰竭性缺口（抄底信号）

#### 9. 连板股开板买点 [29:08]
- 连续涨停第一次开板，换手率30%-50% + 当天回封 = 接力买点，第二天有溢价
- 换手率>60% + 未回封 = 出货，不要买

#### 10. MACD背离信号 [40:52]
- 顶背离：股价创新高但MACD红柱和DIF未创新高，高位两次顶背离几乎100%下跌
- 底背离：股价创新低但MACD绿柱和DIF未创新低，低位两次底背离上涨概率>80%

#### 11. 主升浪启动信号 [44:18]
- 长期盘整后连续三个涨停板 + 成交量逐步放大 = 主升浪启动
- 第三个涨停后回踩可买入，后续至少还有30%涨幅

#### 12. 涨停后十字星含义 [27:25]
- 上涨中期缩量十字星 = 洗盘，继续持有
- 高位放量十字星 = 出货，要卖出

### 四、交易规则与经验总结

#### 板块龙头选股规则 [37:50]
- 同板块最先涨停、封单最大、涨幅最高的就是龙头
- 买股优先买龙头，龙头涨多跌少有溢价，跟风涨少跌多

#### 止盈止损设置 [42:26]
- **止损**：买入价-8%必须卖，不能扛单
- **止盈**：短线设置15%-20%到点止盈，或沿5日均线跌破再走；中长线沿20日均线跌破再止盈
- **案例** [45:21]：2023年AI板块剑桥科技从10元涨到70元，每次回调未跌破20日均线，沿20日均线持有吃完整波主升浪

#### 讲师核心总结

> "所有的技术分析都是概率，没有100%正确的，所以一定要做好止损，把亏损控制在小范围，赚的时候赚大钱，长期下来你就是盈利的，不要追求每次都对，要追求亏小赚大。" [46:29]
