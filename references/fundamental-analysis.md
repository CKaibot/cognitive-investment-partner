# 基本面分析：六维透视框架

本文档为 G2 在个股/财报模式下提供基本面分析的深度展开。技术面有四份参考文档兜底，这份文档给基本面同等待遇。

## 一、生意本质

目标：用一句话让外行听懂这家公司怎么赚钱。

**核心提问**：
- 收入从哪来？靠卖产品、卖服务、还是赚利差/佣金？
- 护城河是什么？品牌、规模、网络效应、转换成本、牌照/特许经营？
- 护城河在加深还是变浅？五年前的护城河现在还在吗？
- 这个生意有没有"飞轮"——收入增长是否会自我强化？

**常见陷阱**：
- 照抄公司官方话术（"致力于成为行业领先"没有信息量）
- 把"业务描述"当成"商业模式"（前者是做什么，后者是怎么赚钱）
- 忽略商业模式随时间的变化（很多公司的赚钱方式已经跟上市时不同了）

## 二、财务健康度

四个层次，逐层深入：

### 2.1 收入质量
- **主业 vs 非经常性损益**：扣非净利润才是真实盈利能力。一次性收益（卖资产、政府补贴、投资收益）拿掉后还剩多少？
- **收入确认激进度**：应收账款增速是否远超收入增速？（如果是，可能在提前确认收入）
- **客户集中度**：前五大客户占比超过50%是高风险信号

### 2.2 利润含金量
- **经营性现金流 vs 净利润**：长期低于1意味着利润"赚到了但没收到钱"
- **自由现金流**：经营现金流 - 资本开支。这才是公司真正可以分配给股东的钱
- **现金转化周期**：从花钱到收钱要多久？越短越健康

### 2.3 资产负债表的"真话"
- **应收账款**：增速远超收入 → 可能放宽信用条件冲收入
- **存货**：增速远超收入 → 产品可能滞销
- **商誉**：占净资产比例过高 → 暴雷风险（关注减值测试）
- **有息负债率**：利息支出 / 经营利润 > 30% 需警惕
- **表外负债**：担保、应付票据、租赁负债——报表上不显眼但真要还钱

### 2.4 关键比率
| 指标 | 看什么 | 红线 |
|------|--------|------|
| ROE | 持续性（连续3年>15%是好生意） | 靠高杠杆堆的ROE要打折 |
| 毛利率 | 趋势比绝对值重要 | 持续下降=竞争加剧或成本失控 |
| 净利率 | 与同行比 | 远高于同行需要解释为什么 |
| 自由现金流 | 正/负及趋势 | 连续3年为负需要好的理由 |
| 资产负债率 | 行业差异大 | 重资产行业50-60%正常，轻资产>40%需警惕 |

## 三、估值参照

估值不是算出一个"准确价格"，而是判断"市场隐含了什么预期"。

### 3.1 历史纵向
- 当前 PE/PB 在自身历史分位数的位置（近5年/10年）
- 历史上类似估值水平时，后续表现如何
- 注意：行业周期股（钢铁、化工等）不能看PE——低PE可能是景气高点

### 3.2 同行横向
- 相对同行是溢价还是折价？溢价是否合理（增速更快？质量更好？）
- 注意业务结构差异——看似同行但商业模式不同，估值不可比

### 3.3 隐含增长逆向推导
- 当前股价隐含了多少年、多快的增长？
- 这个增长假设合理吗？历史上有多少公司真的做到了？
- **核心问题**：如果增长不达预期，当前价格的安全边际有多少？

## 四、风险地图

### 4.1 已知风险
- 行业政策风险（监管收紧、补贴退坡）
- 竞争格局变化（新进入者、替代品）
- 管理层风险（关键人物依赖、治理问题）
- 宏观敏感度（利率、汇率、大宗商品价格）

### 4.2 分析失效条件
这是最重要的部分——**什么情况下你的整个分析逻辑会作废？**
- 如果核心假设（增速、利润率、市场份额）不成立
- 如果行业发生结构性变化（技术颠覆、政策转向）
- 如果出现了你没考虑到的风险（黑天鹅清单）

### 4.3 最坏情景
- 如果上述风险同时发生，下行空间是多少？
- 这个下行你能承受吗？（与仓位大小挂钩）

## 五、技术面速览

此维度在个股模式下为速览，详细分析由技术面参考文档支撑。

速览要回答的问题：
- 当前趋势状态（上升/下降/盘整），20日均线方向
- 近期有无关键K线信号或中大级别形态
- 量价配合是否健康
- 如有明确技术买卖点，给出关键价位
- 个股独立性：跟随板块/指数还是独立运作？
- 跨市场信号：同行业港股/美股走势参考

## 六、认知偏差扫描

此维度由 G3 驱动，重点检测用户问题中暗含的偏差信号。

个股分析中最常见的偏差：
- **锚定效应**：用户反复提及买入价或历史高点（"从100跌到50了，很便宜"——便宜不便宜跟你的买入价无关）
- **确认偏差**：用户只提利好不提利空（"这公司前景很好"——问问他有没有认真找过反面证据）
- **禀赋效应**：已持有的股票更"宽容"（"长期看好"是因为真看好，还是因为已经买了？）
- **过度自信**：对预测过于确定（"明年肯定翻倍"——用概率语言替换确定性语言）

## 决策清单

每次个股分析结束后，引导用户自检：

- [ ] 我能一句话说清楚这家公司为什么赚钱吗？
- [ ] 如果股价再跌30%，我的分析逻辑是否依然成立？
- [ ] 我是否在寻找支持已有结论的证据？
- [ ] 这笔投资在整体仓位中占比合理吗？
- [ ] 技术面是否支持现在入场？
- [ ] 我能说清楚"什么情况下我会卖出"吗？
