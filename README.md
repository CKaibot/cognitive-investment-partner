# 认知投资决策伙伴 (Cognitive Investment Partner)

一个为 [Claude](https://claude.ai) 设计的投资分析 Skill，融合行为金融学、认知科学和实战技术分析体系，帮助个人投资者做出更理性的投资决策。

## 核心能力

- **认知偏差检测**：在每次分析中自动识别锚定效应、确认偏差、损失厌恶等 15 种常见投资认知偏差
- **个股六维透视**：生意本质 → 财务健康度 → 估值参照 → 风险地图 → 技术面速览 → 认知偏差扫描
- **技术面研判**：K线形态、成交量分析、均线体系、中大级别形态学（W底/头肩/三角形等）
- **投资组合评估**：集中度、相关性、风险预算、仓位纪律检查
- **市场研究**：行业趋势、板块分化、跨市场映射分析
- **决策辅助**：事前验尸、反转测试、技术面时机判断

## 市场覆盖

- **A股和港股**：主要分析市场
- **美股**：重要参考（估值锚、行业趋势、资金流向）

## 技术分析体系

Skill 内置了一套经过实战验证的技术分析参考文档：

| 文档 | 内容 |
|------|------|
| `kline-technical-analysis.md` | K线见底/见顶/突破形态、交易纪律 |
| `volume-analysis.md` | 成交量量化标准、量价关系、主力行为识别 |
| `moving-averages.md` | 均线体系、均量线配合、短线战法 |
| `chart-patterns.md` | 中大级别形态（W底、头肩、三角形等）、MACD背离、缺口理论 |
| `market-structure.md` | 个股/板块/指数分化逻辑、选股优先级 |
| `cognitive-biases.md` | 15种投资认知偏差详解与应对策略 |

## 安装方法

将本仓库克隆到 Claude 的 skills 目录：

```bash
# 方法一：克隆到用户 skills 目录
git clone https://github.com/<your-username>/cognitive-investment-partner.git ~/.claude/skills/cognitive-investment-partner

# 方法二：手动复制
cp -r cognitive-investment-partner/ ~/.claude/skills/cognitive-investment-partner
```

安装后，在 Claude 对话中提到任何投资相关话题即可自动触发。

## 5 种工作模式

1. **个股/财报分析**（六维透视框架）
2. **投资组合评估**（集中度 + 相关性诊断）
3. **市场研究/趋势分析**（板块分化 + 跨市场映射）
4. **决策辅助**（事前验尸 + 反转测试 + 偏差扫描）
5. **技术面研判**（K线 + 量价 + 均线 + 形态学）

模式根据用户问题自动切换，无需手动选择，可组合使用。

## 核心原则

- **有依据地给建议** — 给出明确操作思路，说清依据和失效条件
- **位置比形态重要** — 同样的K线形态在不同位置意义完全不同
- **亏小赚大** — 止损纪律是生存的底线
- **偏差检测贯穿始终** — 融入每一步分析，不是附加模块
