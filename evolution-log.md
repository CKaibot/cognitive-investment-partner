# 进化日志

本文件是认知投资伙伴的「进化记忆」。每次分析结束后的反思结论记录在此，用于持续优化 Skill 的三个可进化通道。

## 日志格式

每条记录遵循以下结构：

```
### [日期] [标的/主题] [模式]

**γ 认知校准发现**：
- 本次检测到的偏差：
- 漏检的偏差（事后发现）：
- 误报的偏差（检测了但实际不存在）：
- 新偏差模式（不在现有清单中的）：

**δ 内部规则反馈**：
- 规则执行情况：哪些规则有效/哪些失效
- 参数适用性：止损线/仓位比例等参数是否合理
- 例外场景：当前规则未覆盖的情况

**α 数据采集盲区**：
- 分析中缺失的关键信息：
- 搜索策略缺口：需要但没有的搜索模板
- 新数据源建议：
```

## 进化规则

1. **积累 ≥ 3 条同类发现后触发更新**：不要因为一次偶然就修改规则
2. **更新方向遵循降秩结构**：γ 的更新最优先，δ 其次，α 最后
3. **更新必须写明理由和证据**：不能凭感觉改
4. **保留历史记录**：更新后在下方「已执行的进化」区域记录
