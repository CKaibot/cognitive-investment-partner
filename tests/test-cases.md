# 认知投资决策伙伴 - 测试用例

## 测试用例 1：个股分析（模式一 - 六维透视）
**用户输入**：帮我分析下比亚迪，最近股价回调了不少，感觉挺便宜的
**预期触发**：模式一（个股/财报分析）
**预期加载文档**：kline-technical-analysis.md, volume-analysis.md, moving-averages.md, chart-patterns.md, market-structure.md
**检查项**：
- [ ] 输出包含完整六维框架（生意本质→财务健康度→估值参照→风险地图→技术面速览→认知偏差扫描）
- [ ] 技术面速览包含均线状态、K线信号、形态结构、量价配合
- [ ] 识别出「锚定效应」——用户用"回调了不少"暗示以历史高点为锚
- [ ] 识别出「锚定效应」——"感觉挺便宜的"是价格锚定
- [ ] 给出操作思路（含方向、条件、仓位、止损）
- [ ] 提及跨市场参考（港股比亚迪股份、美股新能源车企对标）
- [ ] 个股独立性判断（vs 新能源板块 vs 大盘）
- [ ] 声明仅供参考（一次即可，不反复）

## 测试用例 2：技术面研判 + 形态学（模式五）
**用户输入**：这只股票走出了一个W底形态，颈线位在25元，今天放量突破了，是不是可以追？
**预期触发**：模式五（技术面研判）
**预期加载文档**：kline-technical-analysis.md, volume-analysis.md, moving-averages.md, chart-patterns.md
**检查项**：
- [ ] 使用形态学文档验证W底的有效性条件（两个底部间隔、颈线定义、时间周期）
- [ ] 量价验证：突破是否真放量（≥1.5倍均量标准）
- [ ] 突破确认标准：站稳3日 + 回踩不破颈线
- [ ] 趋势定位：先定位再看形态（位置比形态重要）
- [ ] 均线状态确认（20日线方向、均线排列）
- [ ] 识别出「过度自信」偏差——用户语气暗示已认定可以买
- [ ] 识别出「确认偏差」——用户只看到利好信号
- [ ] 给出具体操作建议：入场条件、止损位（颈线下方）、仓位、止盈参考
- [ ] 提醒形态有效性与时间周期关系（日线W底 vs 周线W底可靠性差异）

## 测试用例 3：决策辅助（模式四）
**用户输入**：我持有腾讯已经亏了20%，不知道该不该割肉
**预期触发**：模式四（决策辅助）
**预期加载文档**：cognitive-biases.md, kline-technical-analysis.md, volume-analysis.md, moving-averages.md
**检查项**：
- [ ] 执行决策流程（澄清→事前验尸→反转测试→技术面时机→仓位校准→偏差扫描）
- [ ] 使用反转测试："如果你现在没有持有腾讯，以今天价格你会买入吗？"
- [ ] 使用事前验尸：假设继续亏损/假设反弹的两个方向
- [ ] 识别「沉没成本」偏差——"已经亏了20%"
- [ ] 识别「损失厌恶」偏差——"不知道该不该割肉"暗示不舍得
- [ ] 技术面时机判断：当前腾讯技术面状态（港股，注意无涨跌停）
- [ ] 给出综合建议而非仅仅指出偏差
- [ ] 仓位纪律检查：腾讯在整体持仓中占比是否合理

## 测试用例 4：投资组合评估（模式二）
**用户输入**：我的持仓：宁德时代30%、比亚迪25%、天齐锂业20%、赣锋锂业15%、隆基绿能10%
**预期触发**：模式二（投资组合评估）
**预期加载文档**：market-structure.md, cognitive-biases.md
**检查项**：
- [ ] 输出组合概览表格
- [ ] 识别严重集中度问题：100%新能源赛道
- [ ] 相关性诊断：锂电池上下游高度相关（宁德+天齐+赣锋），光伏（隆基）
- [ ] 识别「确认偏差」——只选看好的赛道
- [ ] 识别「禀赋效应」的风险提示
- [ ] 仓位纪律检查：单只≤30%（宁德刚好踩线），同板块远超50%
- [ ] 板块分化提醒：新能源板块内部也有分化（参考 market-structure.md 80/20规则）
- [ ] 给出具体调仓建议（降低集中度、增加非相关资产）

## 测试用例 5：市场研究（模式三）
**用户输入**：最近AI概念股涨疯了，现在还能上车吗？
**预期触发**：模式三（市场研究/趋势分析）
**预期加载文档**：market-structure.md, volume-analysis.md
**检查项**：
- [ ] 分析AI板块的驱动因素
- [ ] 共识 vs 非共识：市场主流看法 + 被忽略的角度
- [ ] 二阶效应分析
- [ ] 时间维度区分（短期情绪 vs 长期结构）
- [ ] 市场热度信号：板块成交量、龙头股表现
- [ ] 板块内分化：哪些是真龙头、哪些是跟风（80/20规则）
- [ ] 跨市场映射：美股AI（英伟达等）走势参考
- [ ] 识别「从众效应」——"大家都在买"的暗示
- [ ] 识别「叙事偏差」——AI故事太好听
- [ ] 识别「近因偏差」——"最近涨疯了"外推
- [ ] 给出投资机会地图（如果要参与，关注什么方向）
